Bienvenido al Frontend Usuario

Esta aplicacion consume una API de base de datos masivas que se ejecuta en Docker. El frontend esta desarrollado en Flask y funciona como proxy para evitar problemas de CORS.

Modulo Mundo

Modulo completamente funcional que permite consultar informacion de paises, estados y ciudades. Incluye funcionalidades para cargar todos los datos, filtrar por nombre de pais y encontrar ciudades repetidas.

Funcionalidades disponibles:

  • Cargar todos los paises, estados y ciudades
  • Filtrar informacion por nombre de pais especifico
  • Listar ciudades repetidas por nombre de pais
  • Verificar estado de la API (Health check)
  • Paginacion de resultados (100 por pagina)
  • Busqueda insensible a mayusculas/minusculas
  • Visualizacion simplificada (pais, estado, ciudad)

Instrucciones de Ejecucion

1. Crear entorno virtual

python -m venv venv

2. Activar entorno

Windows: venv\Scripts\activate
Linux/Mac: source venv/bin/activate

3. Instalar dependencias

pip install -r requirements.txt

4. Configurar variables (opcional)

Copiar .env.example a .env y ajustar API_BASE

5. Ejecutar aplicacion

python app.py

6. Abrir en navegador

http://localhost:5000

Informacion de la API

La API debe estar ejecutandose en Docker en http://localhost:8080 para que el frontend funcione correctamente.

El frontend actua como proxy y todas las llamadas se realizan a traves de la ruta /api/ para evitar problemas de CORS.
